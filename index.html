What is Python?

Python is a high-level, interpreted, and general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically-typed and garbage-collected. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Python is widely used for web development, data analysis, artificial intelligence, scientific computing, and more. Its extensive standard library and the availability of third-party modules make it a versatile language.

The “Hello, World!” program is the first step towards learning any programming language. In Python, it can be written as:

print("Hello, World!")
Data Types

Python supports various data types, including:

  • Numbers: int, float, complex
  • Strings: str
  • Lists: list
  • Tuples: tuple
  • Dictionaries: dict
  • Sets: set
Control Flow

Python provides several control flow tools to manage the order in which your code is executed:

  • Conditional Statements: if, elif, else
  • Loops: for, while
  • Comprehensions: list, dictionary, set comprehensions
  • Exception Handling: try, except, finally
Functions

Functions are defined using the def keyword. They are a way to encapsulate code into reusable blocks. 

def greet(name):
	print(f"Hello, {name}!")

greet("Alice")
Modules

Modules in Python are files containing Python code. They can define functions, classes, and variables that you can include in your project using the import statement. 

# math module
import math

print(math.sqrt(16))
Object-Oriented Programming

Python supports Object-Oriented Programming (OOP) with classes and objects. OOP is a programming paradigm based on the concept of "objects", which are data structures that contain data and methods.

Class: A blueprint for creating objects. It defines a set of attributes and methods that the created objects will have.

Object: An instance of a class. It contains the data and methods defined by the class. 

class Dog:
	def __init__(self, name, age):
		self.name = name
		self.age = age

	def bark(self):
		print(f"{self.name} says woof!")

dog1 = Dog("Buddy", 3)
dog1.bark()
